These are example PowerShell scripts to install Microsoft 365 Apps Monthly Enterprise x64 with an Microsoft Intune WIN32 package.
The XML file contains MigrateArch="TRUE to migrate the architecture from 32-bits to 64-bits.

The repo contains an install, uninstall and detection script.

For more information about these scripts read the related [Blog Post](https://www.inthecloud247.com/)
